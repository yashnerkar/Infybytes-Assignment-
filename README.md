


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```
Start the client

```bash
  npm run start
```
Start the server

```bash
  npm run dev
```




## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express,Socket.io
